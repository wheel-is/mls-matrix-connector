# Security

- This is an unofficial private-use connector for an authorized MLSListings account.
- MLS credentials and MFA are entered only into the visible MLS website.
- On Windows, cookies and Matrix form state are encrypted for the current Windows user with DPAPI.
- The installer verifies the versioned connector ZIP with a pinned SHA-256 checksum before extraction.
- The installer backs up an existing Claude Desktop configuration before merging its MCP server entry.
- Never upload or share HAR files, browser profiles, session files, cookies, or passwords with this project.
- Search results returned to Claude are sent to Claude for processing.
- Confirm that use is permitted by the account holder's MLS agreement and applicable rules.
